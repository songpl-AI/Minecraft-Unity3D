✨ 总结
-------

C# 版本：
  核心：2个文件
  文档：4个Markdown
  示例：3个完整示例
  目标：5分钟集成到任何Unity项目

Lua 版本（xLua）：
  核心：1个 Lua 文件
  示例：1个 Lua 示例 + 1个 C# 集成脚本
  文档：2个 Markdown
  特性：支持热更新、动态加载
  性能：约为 C# 版本的 40-60%

选择建议：
  - 需要最高性能 → 使用 C# 版本
  - 需要热更新、灵活配置 → 使用 Lua 版本
  - 两者可以并存，根据场景选择
